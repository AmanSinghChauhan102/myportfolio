Stack Data Structure Notes

Stack Overview

A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle.

Stack Operations

1. Push: Adds an element to the top of the stack.
2. Pop: Removes the top element from the stack.

C Implementation

#include <stdio.h>
#define MAX 5  // Maximum size of the stack

int stack[MAX];
int top = -1;  // Initialize top index to -1

// Push operation
void push(int element) {
    if (top < MAX - 1) {
        top++;
        stack[top] = element;
    } else {
        printf("Stack Overflow\n");
    }
}

// Pop operation
int pop() {
    if (top >= 0) {
        int element = stack[top];
        top--;
        return element;
    } else {
        printf("Stack Underflow\n");
        return -1;  // Return a sentinel value to indicate error
    }
}

int main() {
    push(1);
    push(2);
    push(3);

    printf("Popped element: %d\n", pop());  // Output: 3
    printf("Popped element: %d\n", pop());  // Output: 2
    printf("Popped element: %d\n", pop());  // Output: 1

    return 0;
}

Explanation

  • We define a stack array of size MAX and a top variable to keep track of the top index.
  • The push function adds an element to the top of the stack if there is space available.
  • The pop function removes the top element from the stack if it is not empty.

Example

Imagine a stack of books:

  • When you add a book, it goes on top (push).
  • When you remove a book, it comes from the top (pop).

In this example, we push three elements (1, 2, and 3) onto the stack and then pop them off, demonstrating the LIFO principle.
